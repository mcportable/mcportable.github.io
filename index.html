How does it work?

Basically it based on how does Minecraft run. Minecraft is coded on Java, so just run it using Java. In Minecraft Launcher, you will need to press "Play" button to play. The launcher will run a Java command to run Minecraft located in AppData. We just take the Minecraft files there and make a small batch file to run it portable. Here is a command that we use to run Minecraft:
java -Xms512m -Xmx1g -Djava.library.path=natives/ -cp "minecraft.jar" net.minecraft.client.Minecraft

Some Java libraries are required to run Minecraft, they are LWJGL. "LWJGL is a Java library that enables cross-platform access to popular native APIs useful in the development of graphics (OpenGL, Vulkan), audio (OpenAL) and parallel computing (OpenCL) applications. This access is direct and high-performance, yet also wrapped in a type-safe and user-friendly layer, appropriate for the Java ecosystem." - LWJGL Homepage. The game won't work if we don't let the game use the libraries. So, in the run command, we have to integrate libraries by change the command from "minecraft.jar" to:
"minecraft.jar;libs\lwjgl\2.9.0\lwjgl-2.9.0.jar;libs\lwjgl_util\2.9.0\lwjgl_util-2.9.0.jar"
:: Minecraft 1.5.2 uses LWJGL 2.9.0

Some parameters are needed, too. For example, -username or -uuid is needed to login to offline account (Online account coming soon). Add these parameters at the end of the command.
